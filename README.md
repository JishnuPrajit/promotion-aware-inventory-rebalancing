# Promotion-Aware Inventory Rebalancing for a Multi-Store Retail Network

## Project Overview
This project aims to build a price- and event-aware inventory rebalancing system for a multi-store retail network using the M5 forecasting dataset. The goal is to forecast item-level demand across stores, identify potential inventory shortages and surpluses, and recommend rebalancing decisions that improve product availability while reducing excess inventory.

## Dataset
This project uses the M5 Forecasting dataset. The main files used are:
- sales_train_validation.csv
- calendar.csv
- sell_prices.csv
